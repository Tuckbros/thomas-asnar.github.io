À propos de ce site

J'ai migré ce site sur les pages Github. Ainsi je m'affranchis des problèmes d'hébergement, de versioning, des sauvegardes, et du nom de domaine. Désormais, c'est simple comme bonjour ; je git push mes datas et, le site est mis à jour. Petite anecdote qui peut avoir quelques conséquences sur la visibilité du site : mon ancien nom de domaine (comme beaucoup de domaines persos) étaient bloqué par les proxys des entreprises ; GitHub non.

Vous trouverez les sources sur mon GitHub

J'utilise les technologies suivantes :

  • Jekyll pour mes templates et la génération du site. Le système de Blog viendra après, quand j'aurais migré mes posts Wordpress
  • Toutes mes données sont en .json et traiter par les templates Liquid de Jekyll 
    #Exemple _data/moi.json
    {
    nom: "ASNAR",
    prenom: "Thomas",
    metier: "Analyste d'exploitation",
    anniversaire-j: "19",
    anniversaire-m: "01",
    anniversaire-a: "1983",
    permis: "permis B",
    ...
    ...
    ...
    }
  • Grunt ou des gems dans le fichier Gemfile, pour automatiser mes tâches de production (concaténer, minimiser les css et les js, mise à jour de GitHub)
  • npm (gestionnaire de packet NodeJS) pour gérer mes dépendences de développement
  • Bower pour gérer mes dépendences de librairies externes (comme jQuery ou boostrap)
  • En front-end, c'est du classique HTML5
